# love is heavenly ♥

*a midnight sanctuary* — a cozy, candlelit personal website and online journal.
Dark yet comforting: deep charcoal and moonlit navy, candleglow, gentle grain,
and soft star-like sparkles. A handcrafted place for writing, reflection,
memories, music, and photographs.

## the rooms

- **home** — a poetic welcome, drifting moonlight, floating candle-lights, and tonight's featured entry
- **journal** — entries as handwritten notes with dates, moods, and tags
- **memory garden** — memories that bloom as glowing flowers and keepsakes you can open
- **music room** — favorite songs displayed like records on a shelf, with embedded players
- **photographs** — a soft photo album with gentle lightbox transitions
- **about** — a customizable profile and a few favorite things
- **guestbook** — a place for visitors to leave kind notes

## built by hand

Pure HTML, CSS, and a little vanilla JavaScript — no build step, no framework.
Slow, subtle animations only; cozy dark mode by default; responsive; and
mindful of contrast, keyboard navigation, and reduced-motion preferences.

## keeping it alive

All content lives in plain, well-commented files in [`content/`](content/).
See **[CONTENT.md](CONTENT.md)** for the full, gentle guide to adding journal
entries, memories, music, photos, and more.

*keep it soft.*
